### :hammer: Installation

Firstly, install dependencies using yarn or npm

```bash
git clone 
cd FeedMe
yarn
npx pod-install
```

### :car: Usage

Start the application

```bash
npx react-native run-ios # for ios
npx react-native run-android # for android

```
